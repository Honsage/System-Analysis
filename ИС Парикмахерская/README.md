# ИС Парикмахерская

Информационная система "Парикмахерская" предназначена для автоматизации процесса управления небольшим предприятием (салоном красоты).

Период проектирования: март – май 2025 г.

## Описание

В документе представлены диаграммы и схемы, построенные в рамках проектирования данной ИС.
